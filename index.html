Species occurrences in nature reserves

Search for a taxon, pick a country or the whole world, and optionally restrict to protected areas — then explore GBIF occurrences overlaid on OpenStreetMap reserve boundaries.

Taxon lookup is powered by the Wikidata API (matching items with a GBIF ID statement, property P846). GBIF data is converted to RDF via gbif_parquet and queried live on QLever.

Built as a demo for the Ebbe Nielsen Challenge 2026.

Queries run across different datasets and regions, so they may take a while. Filtering for the Natural Reserves is a step particularly time-consuming.

Source code: github.com/lubianat/gbif_sparql
